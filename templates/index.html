Welcome to MGApi

MGApi is a RESTful API that allows you to generate mazes quickly and easily!

Generate Maze:

  • /maze/generate/json - Generate a default 10x10 maze with the Recursive Backtracker algorithm (JSON Format)
  • /maze/generate/json/width/height - Generate a width by height maze with the Recursive Backtracker algorithm (JSON Format)
  • /maze/generate/json/strategy/width/height - Generate a width by height maze with the algorithm provided in strategy (JSON Format)

Available Algorithms:

  • Recursive Backtracker
  • Hunt And Kill

Render Maze:

  • /maze/render - Render a default 10x10 maze with the Recursive Backtracker algorithm
  • /maze/render/width/height - Render a width by height maze with the Recursive Backtracker algorithm
  • /maze/render/strategy/width/height - Render a width by height maze with the algorithm provided in strategy
